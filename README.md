# Amrita Sinha — Personal Portfolio

A clean, animated, fully responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Site:** [amritasinha8976.github.io/Amrita.Portfolio](https://amritasinha8976.github.io/Amrita.Portfolio/)

## Features

- **Animated gradient background** that shifts continuously and reacts to cursor movement
- **Cursor glow effect** — a soft radial spotlight that follows your mouse with smooth lerp animation
- **Per-section colour tints** — each section has a distinct warm palette tone so navigation feels natural
- **Glassmorphism cards** with `backdrop-filter` blur on all skill, project, experience and contact cards
- **Typing effect** in the hero section cycling through roles
- **Scroll-triggered fade-in animations** using IntersectionObserver
- **Scroll progress bar** at the top of the page
- **Light / Dark theme toggle** with localStorage persistence — defaults to light
- **Fully responsive** — mobile hamburger menu, stacked layouts on small screens
- **Resume download** — direct PDF download link

cd Amrita.Portfolio
